# Python Desktop Cleaner

## Overview
This script automatically organizes files on the Desktop by grouping them into folders based on file extensions. It reduces clutter and helps keep the workspace clean.

## Features
- Detects file extensions automatically
- Creates folders dynamically (e.g., `PDF Files`, `PNG Files`)
- Moves files into the correct folder
- Uses only Python standard library

## How to Run
```bash
python desktop_cleaner.py
